# Calculadora de IMC

## Descrição

Projeto desenvolvido em React + Typescript<br/>
Consiste em uma calculadora de IMC (Peso/(Altura\*Altura)) onde mostra como resultado o seu IMC e também a qualificação (Magreza, Normal, Sobrepeso e Obesidade).

## Instalação
- `npm install`

## Instalação
- `npm start`
